# Web App (Deprecated)

**Status: Deprecated — not in use.**

This React + Vite + TypeScript web app was built to provide desktop browser access for managing projects and receipts. We've pivoted to a simpler architecture:

- The **Flutter mobile app** (`../mobile/`) is the primary interface
- **Google Sheets** replaces the web dashboard — dad opens the sheet directly on his PC to view/edit data
- No backend or web frontend is needed

This code is kept for reference but is not deployed or maintained.
